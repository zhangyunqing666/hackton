# 场景智能——自动驾驶新范式
随着自动驾驶技术的发展，结合新的大模型技术，自动驾驶将迎来崭新的时代。在这一背景下，我们调研了目前高等级自动驾驶落地困难的原因，并提出了新的自动驾驶范式——场景智能。其包括基于大模型的场景工程、高精度场景仿真和自动驾驶泛在操作系统三部分。我们为每个部分设计了方案和实现流程。希望我们的想法可以为高级别自动驾驶的落地提供参考，非常感谢黑客松大赛给我们展示我们想法的机会。
